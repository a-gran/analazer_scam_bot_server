# Описание проекта

Telegram-бот на `aiogram`: игрок читает сообщения и определяет, скам это или безопасное сообщение.

Проект специально оставлен в плоской структуре: без папок `handlers/` и `game_logic/`.

```
analyzer_scam_bot/
├── bot.py             — точка входа, запуск бота
├── config.py          — токен и игровые константы
├── states.py          — состояния FSM
├── keyboards.py       — inline-клавиатуры
├── data.py            — база вопросов
├── start.py           — обработчик /start и «Играть снова»
├── game.py            — показ вопросов и обработка ответов
├── answer.py          — проверка ответа и итоговое звание
├── session.py         — создание и чтение игровой сессии
├── results.py         — итоговый экран
├── requirements.txt   — зависимости проекта
└── README.md          — описание проекта
```

## Запуск

1. Установить зависимости:

```bash
pip install -r requirements.txt
```

2. Добавить токен бота в `.env`:

```env
BOT_TOKEN=123456:AAF...
```

3. Запустить:

```bash
python bot.py
```
