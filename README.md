# Описание проекта

Telegram-бот на `aiogram`: игрок читает сообщения и определяет, скам это или безопасное сообщение.

Проект специально оставлен в плоской структуре для простоты: без папок `handlers/` и `game_logic/`.

```
analyzer_scam_bot/
├── bot.py             — точка входа, запуск бота
├── config.py          — токен и игровые константы
├── states.py          — состояния FSM
├── keyboards.py       — inline-клавиатуры
├── data.py            — база вопросов
├── start.py           — обработчик /start и «Играть снова»
├── game.py            — показ вопросов и обработка ответов
├── answer.py          — проверка ответа и итоговое звание
├── session.py         — создание и чтение игровой сессии
├── results.py         — итоговый экран
├── requirements.txt   — зависимости проекта
└── README.md          — описание проекта
```

## Локальный запуск через polling

Бот запускается как обычная локальная программа. Webhook, Sourcecraft и отдельный сервер для деплоя не нужны: файл `bot.py` сам подключается к Telegram через polling.

1. Установить зависимости:

```bash
pip install -r requirements.txt
```

2. Создать заготовку бота в Telegram и получить токен:

   1. Открыть Telegram.
   2. В поиске найти официального бота `@BotFather`.
   3. Открыть чат с `@BotFather` и нажать `Start`.
   4. Отправить команду:

   ```text
   /newbot
   ```

   5. Ввести название бота. Это обычное имя, которое будет видно пользователям, например:

   ```text
   Analyzer Scam Bot
   ```

   6. Ввести username бота. Он должен быть уникальным и обязательно заканчиваться на `bot`, например:

   ```text
   analyzer_scam_helper_bot
   ```

   7. Если username свободен, `BotFather` создаст бота и пришлет токен. Токен выглядит примерно так:

   ```text
   1234567890:AAH_example_token_text
   ```

   8. Токен нужно сохранить в `.env`. Никому не отправляйте токен и не выкладывайте его в открытый доступ: по нему можно управлять ботом.

3. Добавить токен бота в `.env`:

```env
BOT_TOKEN=123456:AAF...
```

4. Запустить:

```bash
python bot.py
```

В VS Code можно открыть папку проекта, выбрать Python-интерпретатор с установленными зависимостями и запустить `bot.py`.

При запуске бот удаляет старый webhook в Telegram и начинает получать сообщения через polling.
